📌 System Components Overview

🎨 Frontend (React + Vite)
Handles user input, component rendering, and visualization of results. Communicates with backend via REST API.
🌐 Network Layer
HTTP POST requests to /api/simulate or /api/compare. Returns JSON with results and metrics.
⚙️ Backend (Node.js + Express)
Processes requests, validates inputs, selects and executes algorithms, computes metrics.
🔄 Algorithms (4 Types)
FCFS, SSTF, SCAN, C-SCAN - Each implements disk head movement logic independently.
📊 Metrics Engine
Calculates performance metrics: seek time, average time, variance, standard deviation, throughput.
📈 Visualization
Displays results in single or comparison view with interactive charts and metric cards.
